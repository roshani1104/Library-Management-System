# Library Management System

A **Java console-based library management system** using JDBC and MySQL.  
This project demonstrates CRUD operations for books, members, and transactions, along with proper DAO design patterns.

---

## Features

- Add, update, delete, and view books
- Add, update, delete, and view members
- Issue and return books (transaction management)
- JDBC connection management using DBUtil
- Input validation and exception handling
- Clean DAO-Service structure for maintainability

---

## Technologies Used

- Java 
- MySQL
- JDBC
- Git & GitHub
- Eclipse IDE
